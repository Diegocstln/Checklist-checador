# Checklist checador

Web local para evaluar proveedores de checador contra los requerimientos del documento `PROYECTO1_V4.pdf`.

## Que incluye

- Checklist por proveedor.
- Matriz requisito por requisito y checador por checador.
- Estados de cumplimiento: `Cumple`, `Parcial`, `Por validar`, `No cumple`.
- Ranking ponderado por prioridad.
- Filtros por area, prioridad y busqueda.
- Fuentes publicas usadas para sustentar la evaluacion.

## Uso

Abre `index.html` en el navegador o publica el repositorio con GitHub Pages.
